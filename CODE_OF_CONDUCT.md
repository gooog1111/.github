# Code of Conduct

## Our Pledge

We are committed to keeping project discussions respectful, constructive, and useful for everyone.

## Expected Behavior

Please:

* be respectful and patient;
* discuss ideas, code, and documentation constructively;
* provide enough context when reporting issues;
* accept that maintainers may decline requests or close issues that are out of scope.

## Unacceptable Behavior

Unacceptable behavior includes harassment, insults, threats, spam, discrimination, publication of private information, and repeated disruptive comments.

## Enforcement

Maintainers may edit or remove comments, close issues, block users, or take other reasonable action to protect the project space.

## Reporting

If you see behavior that violates this code of conduct, contact the repository owner privately through the contact methods available on the GitHub profile.
